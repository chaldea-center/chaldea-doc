# Laplace

<a href="#">事象記録電脳魔・ラプラス</a>

Laplace 主要负责战斗模拟的功能。欢迎测试反馈！

- 可以在主界面主动选择下方 Laplace 按钮进入
- 也可以点击关卡具体配置旁的计算器按钮进入

太长不看：选人，选礼装，点技能出卡就完事了。目前主要还是周回，高难可以试着用自定义技能发动模拟。
